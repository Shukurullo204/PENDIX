Это платформа нужна чтобы обмениваться сообщениями у проекта есть приложение типа ads, categories, chat, favorites, reviews, users
ads это придложение которое отвечает за то чтобы создавать объявление
categories это придложение которое отвечает за то чтобы можно было создавать категории и подкатегории (вложенные категориями)
chat это придложение сделан чтобы люди могли разговаривать друг с другом и это не просто чат а P2P чат 
favorites это придложение которое отвечает за то чтобы можно было закинуть в избранное или сделать избранным
reviews это придложение которое создает рейтинг для всех пользователей и любой пользователь может поставить рейтинг продавцу 
если хотите вклюючить функцию эту функцию то подключите к html файлам
users это придложение профиль там аватар номер и оъявление пользователя
